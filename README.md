This Weather app is a lightweight and user friendly weather application built with HTML, CSS, and JavaScript. This project aims to provide users with quick and accurate weather information for any location, ensuring a seamless and visually appealing experience.

Key features:
1. Real-time Weather Data: WeatherHub fetches real time weather data using a reliable API, providing users with up to date information on temperature, humidity, wind speed, and more.
2. Intuitive User Interface: The app boasts a clean and intuitive user interface designed with HTML and CSS. Users can easily navigate through the app to access weather details for specific locations.
3. Location Search: Users can search for weather information in any location by entering the city name. The app dynamically updates the weather details based on the user's input.
4. Weather Icons: To enhance the user experience, WeatherHub includes visually appealing weather icons that represent different weather conditions, making it easy for users to understand the current weather at a glance.

Technologies Used:
HTML5: for the structure of the web page.
CSS3: for styling and layout.
JavaScript: for fetching and displaying real time weather data.

Weather API used : https://openweathermap.org/current
